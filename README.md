# MyRecipes Web Application
## Description:
MyRecipes is my first web app that allows users to view and share recipes with each other. 

Demo Link: https://my-recipess-app.herokuapp.com/

## Technologies Used:
- Express
- NodeJS
- MongoDB
- EJS
- Bootstrap
- JavaScript
- CSS
